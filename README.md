# Portfolio-Generator
A modern, interactive portfolio generator built with React and Vite that helps developers and professionals create stunning, personalized portfolio websites instantly.

## Features

- **Multiple Career Templates**: Choose from Web Development, Data Science, Cybersecurity, UI/UX Design, and more
- **Pre-written Content**: Professionally crafted descriptions, taglines, and project examples for each career path
- **Customizable Skills**: Add your own skills or use suggested ones for your field
- **Project Showcase**: Highlight your best work with customizable project cards
- **Responsive Design**: Looks great on all devices
- **Animated Background**: Beautiful particle animations for visual appeal
- **One-Click Export**: Generate and download your portfolio instantly

##  Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/namanmishra20061-gif/Portfolio-Generator.git
cd portfolio-generator
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

## Usage

1. **Select a Career Path**: Choose from available templates (Web Dev, Data Science, Cybersecurity, Design, etc.)
2. **Customize Content**: Edit the pre-filled descriptions, skills, and projects to match your experience
3. **Add Personal Details**: Include your name, contact information, and social links
4. **Preview & Export**: See your portfolio in real-time and download the generated code

## Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Custom CSS animations
- **Build Tool**: Vite

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
